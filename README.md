- Commands
- Covid Tracking
- Bored
- Notes
- Hi to say Hi
- Ping Pong
- 24/7 Work

## Technology

- Node.js
- Express
- Socket io

- Axios
- node-fetch

- http
- fs
- express-fileupload
- express-validator
- path

## How To Use This Repo
1. git clone https://github.com/adityakumar48/wa-bot-v4
2. open Terminal and Type **npm install**
3. Then type **npm start**
4. Now Open Browser and type **localhost:8000**
5. Then Scan **QR CODE**
6. After Scan there started üëç
